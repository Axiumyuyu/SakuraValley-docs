# 基本共识

在这个世界里，我们认为价值的本质是你所投入的不可替代的真实时间与劳动。
为了降低物物交换的摩擦，我们引入了最基础的货币单位：pt/pts。它仅仅是润滑交易的工具，而非你在这个世界生存的终极目标。

## 财富的基石与底线

获取 pts 的方式有很多，但它们遵循着严格的物理与劳动法则。

- 挂机池： 我们理解并充分尊重每个人的现实生活。如果你无暇探索，但仍想在这个世界保留一丝呼吸，你可以通过 `/toafk` 前往挂机池。挂机池给予的报酬仅够维持你最基础的开销，而不应该是你闲暇时的优先选择

挂机池的奖励来源于在场和陪伴，而非不劳而获。我们鼓励通过真实的劳动来获取物资与财富。

## 服务器商店

商店现在按功能被拆分为几个不同的部分，散落在世界各处

输入 `/b-<suffix>` 以购买一些物品，其中suffix的值可以是：

- `dye` 染料及可染色方块，服务器中所有可染色方块均支持互染

- `woodstone` 木石建材，包含了木类，菌柄，蘑菇柄，以及所有带有“石头”性质的建材

- `other` 其他建材，包括了无法被分类到上述类别，但在建筑中常用的其他方块

- `perm` 各类便捷权限和物品

输入 `/s-<suffix>` 以出售一些物品，其中suffix的值可以是：

- `crop` 农作物初级产品

- `ore` 矿石原矿

- `ani` 畜牧业初级产品

- `drop` 各类掉落物

- `sp` 其他无法被分类到上述类别的物品

### 出售

你可以向商店出售的物资:

- 农牧渔业生产的**初级产品**，例如小麦、种子、生牛肉、生猪肉、生鳕鱼、生鲑鱼、皮革、鸡蛋等

- 矿石**原矿**，这是服务器中极为稀有的资源，商店会以极高的价格收购

- 各类掉落物，例如墨囊、线、骨头、羽毛、潜影壳、火药、蜘蛛眼、幻翼膜等

- 部分其他材料，例如紫水晶簇、蜜脾、龙息、生物头颅、各类唱片、海洋之心、犰狳鳞甲、海龟鳞甲、回响碎片等

- 特殊商店中不定期更新的特殊物品，它们可能来自特定结构中的某一种特定产物，或是来自需要制作的某些加工品，这一类收购通常限额较少而且价格较高

### 购买

你可以在商店购买的物资:

- 建筑材料：商店会以非常便宜的价格出售，一方面作为限制生电的补偿，另一方面我们认为，建筑材料不应当成为阻碍创造的障碍

- 染料以及可染色方块：为了方便建筑玩家，所有的可染色方块都支持互染，商店出售染料和所有白色/无色的可染色方块，你可以买到后自行对其染色

- 一些特殊和实用物品但多为一次性物品或者价格不菲，例如快速治疗，快速回城等，还有一些非常强大但也很昂贵的物品，例如风之法杖，它在服务器中的作用相当于一个永久的喷气背包，非常实用

- 生存的便利权限：
> - 随身工作台、随身末影箱
> - 发送彩色文字、在铁砧，书与笔中使用富文本格式
> - 回到传送前的地点，甚至死亡地点的权限
> - 更多的传送点数量上限、更多的商店数量上限
> - 更多的领地数量和范围上限、甚至更便宜的领地租金等

尽管它们大多数是限时的，但通常已经可以极大方便你的日常生存

## 玩家交易与契约

你可以自由地与任何玩家进行交易，无论是通过最原始的丢物品，还是建立你自己的商店。

尽管服务器商店代表了基准线，但我们仍然允许甚至鼓励玩家间自由交易，那些凝结了大量心血的附魔装备、稀有材料和战利品，只有在玩家间的自由贸易中才能发挥其更大的价值。

我们不对玩家间的物价做任何干预。物品的价格，取决于物品中凝结的时间与劳动，以及双方的意愿与契约。请尊重每一笔交易背后的契约精神。

## 财富的终点：留下你的故事

当你的文明发展到极致，积累了大量的财富后，你可以购买一些象征着永恒的特权：

- 永久的便利权限，在前文中提到的，服务器出售的便利权限，你可以选择购买永久，摆脱需要时常续期的困扰。

- 特殊称号与头衔：你可以购买或定制一个特殊的称号与头衔，它将伴随你的一生，并在你的名字旁边显示。

- 时间锚：在岁月和周目的迭代中，少有事物能永存。你可以倾尽毕生积累的财富，为你最引以为傲的核心建筑购买“时间锚”。它将脱离普通领地的生命周期，作为上一代文明的遗迹，永久保留在未来的纪念世界和中心岛中，并可以留下你自己的故事。

> 请记住，再多的财富，都会随时间而消散；财富的终点，并非数字，而是你在这个世界中留下的故事。
