# 起步

欢迎你的到来,在这篇文章里我会用尽可能简短凝练的步骤描述如何在这里起步

1. 加入服务器后,你会出生在五个中心岛之一,这里是是绝对安全的服务器自留地,不允许建造,你还会拥有一个新手领地物品(箱子),能够帮助你快速圈地,以及拥有500pts的初始货币

2. 你需要输入 `/start` 来进行随机传送,这个指令可以无限使用,不消耗任何资源

3. 右键使用新手领地,会在以**你所在的位置**(**注意不是右键点击的位置**)为底面中心,创建一个长宽高均为16的领地(这对于大部分前期情况已经足够),领地id为你的游戏id,之后可以通过 `/res tp <你的游戏id>` 回到这里,也可以自由更改领地范围

4. 在创建完领地后,输入 `/kitlist` 并点击gui领取新手礼包,新手礼包内拥有你前期所需的几乎所有资源,包括木头,工作方块,食物,床,容器,基本工具与基本护甲等等

5. 此时,通常还需要等待一段时间后你才可以前往岛外(可以在侧边栏看到自己的游玩时长,通过 `/sidebar` 打开或者关闭侧边栏显示),这段时间内你可以:

> * 输入 `/s` 前往商店,详细查看商店内有哪些可以收购/出售的物品,以及它们的价格
> * 打开服务器的公告,并详细查看你感兴趣的内容
> * 整理安置你的基地与背包
> * 如果你有朋友和你一起玩,你可能会想和其他朋友共享领地,这是支持的,详细见[领地]()
> * 如果这些步骤都做完了,可以输入 `/toafk` 前往挂机池挂机一段时间,或与其他玩家交流

6. 达到30分钟在线时长后,输入 `/rtp` 前往岛外,这会消耗100pts以及几点饱食度

7. 前往岛外后,通常意味着你已经离开了本服的"新手期",但夜晚的外岛对你来说仍然有不小的危险,因此你可以选择在白天时前往外岛,但地面上也有一些结构不适合前期探索,可以通过小地图等方式自行分辨

## 边缘/意外情况处理

尽管引导已经介绍了你应该做的事,但仍然有极少数未被考虑到的行为或者异常情况

### 在岛屿过渡区/洞穴内受困

输入 `/start`

### 饱食度耗尽且没有食物来源

输入 `/warp spawn3` 回到服务器出生点岛屿,可以缓慢自动恢复饱食度

### 意外死亡

新人加入服务器后有30分钟(*现实时间,等效于1.5游戏日的在线时长*)的死亡不掉落,如果因故超出了30分钟,在完成新手引导前死亡了,请联系我,我不会介入在线且非挂机时长30分钟后的死亡

### 新手领地物品丢失或领地没有成功创建

请输入 `/res tp <你的游戏id>` 确认没有这个领地并且你没有该物品后,联系我

### 无法使用 `/kitlist` 命令或者其他原因导致新手礼包无法领取

请联系我
