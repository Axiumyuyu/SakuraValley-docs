# 起步

欢迎你的到来，在这篇文章里我会用尽可能简短凝练的步骤描述如何在这里起步

1. 加入服务器后，你会以重伤的状态出生在空岛，你还会拥有一个帮助你快速圈地的新手领地物品(箱子)，500pts的初始货币和一个新手礼包，包括你前期所需的几乎所有资源，包括木头、工作方块、食物、床、容器等

2. 你需要输入 `/start` 来进行随机传送，这个指令可以无限使用，不消耗任何资源

3. 右键使用新手领地，会在以**你所在的位置**(**注意不是右键点击的位置**)为底面中心，创建一个长宽高均为 16 的领地(这对于大部分前期情况已经足够)，领地id为你的游戏id，之后可以通过 `/res tp <你的游戏id>` 回到这里,也可以自由更改领地范围

4. 你需要**不穿任何护甲**在床上跳过夜晚或者至少睡 **5** 秒钟来恢复受伤带来的血量上限降低（详见[重伤机制]()）

5. 输入 `/rtp` 前往岛外，前往岛外后，通常意味着你已经离开了本服的"新手期"，但夜晚的外岛对你来说仍然有不小的危险，因此你可以选择在白天时前往外岛，但地面上也有一些结构不适合前期探索,可以通过小地图等方式自行分辨

## 边缘/意外情况处理

尽管引导已经介绍了你应该做的事，但仍然有极少数未被考虑到的行为或者异常情况

### 掉下出生点空岛

输入 `/float`

### 意外死亡

新人加入服务器后有20分钟(*现实时间,等效于1游戏日的在线时长*)的死亡不掉落，如果因故超出了20分钟，在完成[起步](#起步)步骤前死亡了，请联系我

### 新手领地物品丢失或领地没有成功创建

输入 `/res tp <你的游戏id>` 确认没有这个领地并且你没有该物品后，请联系我
