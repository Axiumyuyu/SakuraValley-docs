# 客户端的限制说明

这是一个底层纯净的"Vanilla+"世界,理论上，你可以使用任何原版客户端接入,但我强烈建议你使用群内提供的专用整合包(基于[XPlus整合包Fabric版本](https://modrinth.com/modpack/xplus-2.0-modpack-global)制作,并且加入了服务器所必须的资源包)

如果你使用专用客户端并不对其进行任何修改,你可以不用阅读接下来的部分,如果你希望使用自己的客户端,请务必阅读以下视觉与物理法则：

## 基础材质包:

我增加了一些提升原版游戏的自定义物品，因此你**必须下载并加载群文件内的服务器专属材质包**

如果不这么做,你将无法正确看到自定义的UI菜单,特殊物品的纹理

## 工具的边界：你可以拥有的视野

我们不对优化类模组(如光影,性能提升)做任何限制;除此之外,对于一些通常存在争议的辅助工具,我们的态度是:

- 允许使用矿物透视(X-Ray): 如果你阅读了[世界与环境]()的内容,你就会知道,在这个服务器中不存在"挖矿"的概念,因此自然没有禁止的理由,即使不是如此,我们也认为无意义的盲目挖矿或者炸矿是对创造力以及服务器资源的无意义消耗，只要你遵循经济与资源的共识，透视只是一种节省时间的手段。

- 允许使用灵魂出窍(Free camera): 这是建筑师理应拥有的鹰眼。在规划大型结构或进行精细作业时，你可以自由观察。

- 允许使用建筑投影(Litematica): 欢迎使用投影模组来复刻你的设计蓝图。

## 物理的红线：你绝对不可触碰的领域

我们允许你用更广阔的视野去观察世界,但你必须亲自与这个世界进行物理交互

- 绝对禁止任何突破物理法则与战斗框架的作弊模组(包括但不限于:飞行Flight、杀戮光环Kill Aura、自动寻路或自动操作机器Baritone等)

在这里，工具的意义是帮助你更好地创造，而不是代替你在这个世界生存。任何试图用机器抹杀人类行动过程的行为，都将被视为对这个世界底层秩序的直接挑衅
