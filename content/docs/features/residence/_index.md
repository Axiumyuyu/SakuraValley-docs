# 领地

> 新手的自动圈地道具已为你生成了基础领地。本指南主要涵盖领地建立后的日常高频管理操作。
> 
> 由于领地的操作与用法实在是太多，针对部分不常用操作，你可以自行查看其他帮助文档、游戏内帮助，或者询问。
>
> 服务器使用的插件为 `Residence`， 如果你曾经在其他服务器种使用过类似于 `/res tp` 的指令，那么你可以直接复用那里的知识。

## 创建与删除

当你的初始领地无法满足需求，或者准备建立新的分基地时，你需要手动进行物理选区并创建领地。

选取范围： 手持 木锄，**左键** 敲击一个方块设定 A 点，**右键** 敲击另一个方块设定 B 点。AB 两点构成的对角线将生成一个三维立方体选区。

创建领地： 确认选区后，输入 `/res create <领地名称>`。系统将根据选区体积自动扣除相应的 pt 余额。

删除领地： 输入 `/res remove <领地名称>`，随后按提示输入 /res confirm 或点击聊天栏的提示确认执行。领地一经移除，内部方块将彻底失去保护，且创建时消耗的 pt 不予退还。

## 领地信息查阅

两种方法可任选其一：

- 手持 **线 (String)**，直接左键任意方块，即可在聊天栏快速查看该方块所属的领地名称及所有者信息。

- 站在任意领地内部输入 `/res info`，可查看该领地的边界坐标、体积大小、租金状态以及各项权限开关情况。

## 权限与玩家管理

领地保护的核心在于控制权限（Flags）。权限状态分为三种：`true`（允许）、`false`（拒绝）、`remove`（移除设置，恢复默认）。

> 关于权限的三种状态，我的理解是：
> `true`：默认所有人均可使用
> `false`：默认仅自己可用
> `remove`：不设置/不存在/所有人均不可用
> 
> 这只是大多数权限的情况，特殊权限可能不符合这个理解，可以多摸索几次

### 权限面板：

站在领地内输入 `/res set`，可打开图形化交互界面。通过直观点选，可修改针对所有陌生人的全局权限（例如是否允许进入、破坏、使用容器等）。

### 添加/删除特定玩家

> 有一个预设，用于快速向领地内添加某玩家：`/res padd <玩家ID>`，以及对应的删除命令：`/res pdel <玩家ID>`

通常来说，上面的两个命令已经够用，只有在你想单独更改某人在你的领地内的某项权限时，才需要使用这个指令：`/res pset <领地名称> <玩家ID> <权限名> <状态>`

## 传送与落点设置

输入 `/res tp <领地名称>`，可直接传送到指定领地（前提是目标领地允许传送）。

站在你的领地内，走到你希望设定的精确位置并调整好视角朝向，输入 `/res tpset`。此后，所有传送到该领地的玩家都会准确降落在这个坐标。

## 4. 领地范围更改

如果现有的领地空间无法满足建筑需求，你无需重新圈地，可以站在领地内部直接调整边界（系统会自动计算并扣除/退还差价 pt）：

* **扩大领地：** 视线面朝你想要扩展的方向，输入 `/res expand <格数>`。

* **缩小领地：** 视线面朝你想要收缩的方向，输入 `/res contract <格数>`。

## 5. 领地续租 (Upkeep)

领地需要持续消耗 pt 作为租金来维持底层保护机制。

* **费用扣除：** 系统会根据领地体积，定期从你的账户中扣除对应的 pt 费用。请务必确保你的账户余额充足。

* **手动续租：** 你也可以输入 `/res lease renew <领地名称>`，主动为领地提前缴纳租金，延长保护期限。

* **逾期清理：** 若领地因余额不足导致欠费，在逾期一定时间后，该领地将被系统强制移除，区域内的所有方块将彻底失去保护。
