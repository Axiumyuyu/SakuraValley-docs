# 关于刷怪笼和刷怪蛋机制的草案

## 简介

如同人们在现实中改造生命一样，你也有能力改造这些生物

在发展到后期，你也许会从一些结构的箱子或者宝库中获得一种附魔，它会使你以工具的不可逆损耗为代价，获取某些生物的刷怪蛋

同样的，刷怪笼也不可破坏，除非你拥有了一种特殊的附魔，它使得你能够获得刷怪笼所代表的世界生命之源

## 溯源

如同你使用床设置你的重生点，并且可以无限复活一样，在这个世界中存在了很久的怪物们也有这样的力量，使得自己的生命像玩家的生命一样无穷无尽

刷怪笼，就是远古的怪物们设下的“床”，怪物在这里出生，轮回，死亡，重生，就像玩家一样

它所蕴含的是这个世界的生命力量，但玩家和怪物的这份力量是不同的，无法兼容，你当然不能使用刷怪笼来复活自己，怪物也无法使用你的床

## 机制

### 刷怪蛋

每次使用带有 Pull 附魔的物品击杀某些生物时，将会损耗其默认最大耐久的10%的耐久上限，并掉落一个该生物的刷怪蛋

### 刷怪笼

刷怪笼默认无法破坏，仅可以被带有 Requiem 附魔的合金镐破坏，并掉落一个“生命之源”

除此之外，某些结构的战利品中，也可能包含刷怪笼，它们因为脱离了大地的连接，无力维持自己的物品形式存在，坍缩为了同样的“生命之源”

手持生命之源，被某种指定类型的怪物所攻击时，受到的真实伤害将会转换为生命之源的充能条，如果期间被别的生物所攻击，或者攻击了任何生物，则会重置进度

当生命之源充满能之后，右键同种类怪物，会使其安息，返回怪物自己的原世界，从此终结这幅身体在这个世界的轮回

---

这种仪式不是没有代价的，当你终结一只怪物的轮回时，你是在用你的生命强行扭曲了世界的秩序，因此你的生命上限会永久减少2点

但你获得的，是怪物的认可与慈悲，它们明白了，你并非那些只知道暴力杀戮的玩家

你深知**能力越大，责任越大**

你愿意**作为这个世界中唯一能够思考的生物，负起责任，拯救和引导这些被困的，弱小的存在**

因此，怪物们愿意为你让路，从此时起的30分钟，该类怪物会视你为中立，不是因为它们怕你，而是因为它们明白，世界的秩序应当由像你这样的生物来维护

## 每个人的责任

当你强大到一定程度时，不应该再去无脑砍杀怪物，而是应当负起这个世界中唯一可以思考的生物的责任，去理解、共情，最终产生慈悲，然后怀有敬畏之心地引导它们回到自己原来的世界，消解痛苦

当然，因为世界的秩序难以被彻底扭转，你让这一只怪物脱离轮回，但还有无数的怪物仍然处在轮回之中，怪物的记忆并非永久，它们会逐渐忘记你的所作所为，但至少，你为这个世界做出了贡献，你知道你拯救了它们
