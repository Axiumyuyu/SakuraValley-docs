# 攻击类型与防御类型

## 武器的攻击类型

如果你仔细观察过原版中，武器的攻击方式，你就会意识到，弓和剑的攻击逻辑，是完全不同的；长矛和重锤也是完全不同的，斧头和盾牌又是另一种逻辑

- 长矛：需要积累大量的水平速度，然后将巨大的动能**爆发**出巨大的伤害

- 弓：射出的箭速度很快，而且是穿刺型的，可以**贯穿**目标

- 斧头和重锤：较为类似，它们都通过钝器的**崩解**来震碎目标的内部结构，造成巨大伤害

- 药水：除了物理的攻击方式，我们还有魔法的攻击方式，例如各种伤害药水，它们是**神秘**的，不被常人所理解的

因此，攻击类型被分为这四种：

- `mm:<red>爆发`：长矛

- `mm:<gold>贯穿`：弓

- `mm:<dark_green>分解`：斧头，重锤

- `mm:<blue>神秘`：药水，其他类型的伤害

> 你也许发现了，我没有把剑列在里面，理由同上，剑是一种“基准武器”，它默认没有攻击类型，你可以在后续为其赋予一种类型，也可以直接使用

## 护甲的防御类型

和上文对应，护甲根据其材质的不同，也被分为几种类型：

- 皮革/锁链：它能提供的护甲值不高，并且在原版的护甲材质中属于最贴近实体的，因此它非常**轻**

- 铜/钻石：它们是现实中的硬质材料，因此它们提供的护甲值较高，但是它们非常**重**

- 合金：这是最珍贵的材料，同时也是防御力最高的材料，由多个合金碎片和金合成，有一层一层的**复合**结构

- 金：它是天然的魔法导体，是最**特殊**的材料

因此，防御类型分为这几种：

- `mm:<red>轻装甲`：皮革

- `mm:<gold>重装甲`：铁/钻石

- `mm:<dark_green>复合装甲`：合金

- `mm:<blue>特殊装甲`：金

> 同样的，铁护甲不在其中，理由同上，它们是“基准护甲”，你可以在后续为其赋予一种类型，也可以直接使用

## 属性确定

玩家的攻击属性由手持的物品的攻击属性决定

玩家的防御属性取决于身上穿的所有护甲，根据合成护甲所需的原材料数量计算，一整套护甲需要24个原料合成，如果身上穿的护甲有12个以上的原料由某种类型的护甲制作成，那么玩家当前的防御属性就是该属性

> - 头盔：5
> 
> - 胸甲：8
>
> - 护腿：7
>
> - 靴子：4


如果不能理解的话，可以简单记忆为：

- 穿着三件同属性护甲，类型生效

- 或者至少穿着同类型的头盔与胸甲，类型生效

## 类型克制

每种攻击类型和防御类型都有其短板和长处，以及较优选择，我将它们整理成了一个表格：

| **防御\攻击** | `mm:<red><b>爆发` | `mm:<gold><b>贯穿` | `mm:<dark_green><b>分解` | `mm:<blue><b>神秘` | 常规 |
|:---------:|:------:|:------:|:------:|:------:|:------:|
| `mm:<red><b>轻装甲`   | `mm:<red>WEAK`   | `mm:<gold>EF`     | -      | `mm:<dark_blue>RE`     | -      |
| `mm:<gold><b>重装甲`   | `mm:<dark_blue>RE`     | `mm:<red>WEAK`   | `mm:<gold>EF`     | -      | -      |
| `mm:<dark_green><b>复合装甲`  | -      | `mm:<dark_blue>RE`     | `mm:<red>WEAK`   | `mm:<gold>EF`     | -      |
| `mm:<blue><b>特殊装甲`  | `mm:<gold>EF`     | -      | `mm:<dark_blue>RE`     | `mm:<red>WEAK`   | -      |
| 常规装甲  | -      | -      | -      | -      | -      |



其中，`mm:<red>WEAK` = 1.75 倍伤害，`mm:<dark_blue>RE(RESIST)` = 0.5 倍伤害，`mm:<gold>EF(EFFECTIVE)` = 1.4 倍伤害，-(NORMAL)= 1 倍伤害

## 战斗反馈

在战斗中要你们去时刻记得这一份表格显然是不现实的，尽管它已经非常直觉化，但你仍然不需要去死记硬背上面的表格。

你唯一需要记住的是：**同色的攻击类型和防御类型相互克制，伤害最高**

而表格的其他部分，请相信你的眼睛和耳朵；音效，粒子，热键栏上方的Actionbar也都会给你明确的反馈

## 类型获取与改变

除了没有设定类型的武器和护甲，其他通过合成，战利品等方式获得的武器和护甲，默认都会带有一种攻击类型或防御类型，但是你也可以通过一些方式来改变它们

### 攻击属性

1. 首先获取一个三级的无附魔符文（查看[超限附魔]()页面）
2. 在铁砧上将其与神龟药水（长效或2级）结合，获得属性核心
3. 将属性核心其戴在头上，受到某一种属性的攻击时，会积累知识能量，积累满了之后会转变为带有该攻击类型的属性核心
4. **期间若受到不同属性的攻击，则会重置知识能量**
5. 在铁砧上将武器与带有攻击类型的属性核心结合，可对武器增加或者改变攻击属性，并消耗属性核心

### 防御属性

1. 首先获取一个三级的无附魔符文（查看[超限附魔]()页面）
2. 在铁砧上将其与神龟药水（长效或2级）结合，获得属性核心
3. 使用该属性核心攻击带有某种防御的生物，以积累知识能量，积累满了之后会转变为带有该防御类型的属性核心
4. **期间若攻击了不同防御属性的实体，则会重置知识能量**
5. 在铁砧上将胸甲与带有防御类型的属性核心结合，可对武器增加或者改变防御属性，并消耗属性核心

***注意：以上操作均不可逆，拥有属性的武器和防具无法去掉属性，只能更改***

可以将带有属性的属性核心放在磨石上以重新获得无属性的属性核心

## 自然类型

几乎所有的生物都会带有攻击类型和防御类型中的至少一种，除了一部分被动生物，当鼠标指针指向生物时，会在头顶上显示其攻击类型和防御类型
