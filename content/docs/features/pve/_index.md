# 战斗机制

我对 Minecraft 的 PVE 系统做了一次大更新，并且它是侵略性的，这意味着你必须尝试这个新的战斗系统，但是我将其设计得尽可能自洽和直觉化

## 简要介绍 & 我加入它的原因

传统的 PVE 系统非常无聊，所有人到了后期，穿着一套满配合金甲，拿着合金武器，配上锋利5，到处乱杀，几乎没有什么对手，十分单调和无趣

尽管 Minecraft 中并没有物理系统，但是我们很容易想象到，要玩家穿着这样的护甲，到处跑，甚至还能上天下海，是不可能的，因此我弥补了这部分，护甲拥有一些真实的物理特性，例如沉重的护甲会让你无法飞行，无法游泳，甚至无法快速移动

而对应到武器，我们也很容易注意到，剑，斧，弓，长矛，甚至重锤，它们的攻击方式是完全不同的，但是，它们却造成了同样的伤害，这很显然也是欠缺考虑的，因此我也对它们做了不同的区分与性质特化，有的武器天生适合对付高防御怪物，有的武器适合对付低防御怪物等等

同样我们也可以注意到，不同材质的护甲，带来的不仅是数值上的差异，更有机制上的差异，我们都知道玩家想要在细雪上行走，需要穿上皮革靴子，为什么呢？因为在Minecraft的护甲选择中，皮革是其中最“抗寒”的材质

所以，这个系统由两个部分组成部分，一是对原版护甲数值的重写，二是对武器攻击类型和护甲防御类型的区分
