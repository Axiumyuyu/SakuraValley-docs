# 数值与机制重写

原版的护甲选择过于单一且缺乏有趣的机制，所有人都只是按照皮革-铁-钻石-合金的路径升级自己的护甲，但是原版中还有一些护甲并没有被使用

而护甲本身拥有这么几个主要属性：

- 护甲值：这是护甲的基础数值

- 护甲韧性：它决定了你在受到高伤害时护甲能起作用的程度

- 击退抗性：它会减少你受到的击退

- 最大耐久：在没有经验修补的情况下，它决定了你能打多久的持久战

我们可以回顾一下原版的护甲数值，这里就不多赘述了，但我们可以很容易发现，护甲提供的加成，和合成护甲所需的材料数量，是不对等的，也就意味着当你拥有的材料数量不够制作一整套护甲时，总会有一个性价比最优的选择（即通过最少的材料制作出防御最高的护甲）

因此我重新平衡了这些数值

而只有数值的平衡还是不够的，要让每一种护甲都能发挥其特长，就需要机制的填补

- 皮革和锁链：护甲值相对较低，但具有更高的机动性

- 铜：在现实中铜具有非常好的导电性，因此可以将其扩展到对雷暴的抗性，当被闪电击中时，免疫所有伤害，同时将这些能量转换为正面效果

- 金：在许多小说和虚拟世界设定中，金是天然的魔法导体，金制武器和装备都很容易打上更好的附魔，金的延展性极好，因此我适当提升了它的耐久，金的导热性极好，因此穿着金护甲时，你受到的火焰伤害会减少

- 铁：铁护甲是一套“基准护甲”，考虑到这是一个侵入性的系统，强行让你们去立刻学习这些新的护甲机制（尽管它们已经非常直觉化）是不合理的，如果你暂时不知道如何选择，那么铁护甲就是你的首选

- 钻石：钻石在现实中的硬度和韧性极高，但延展性极差，因此将其制作为护甲，会拥有很高的护甲值，但耐久会比原版更低，并且钻石是晶体，有很高的折射率，因此可以极大地减少守卫者和远古守卫者的激光伤害

- 合金：它拥有最高的防御力，但缺点是它非常重，会极大地降低你的移动速度，穿着时会让你无法起飞，在水中会迅速沉底

---

这些属性不需要你反复查阅这篇文档，一切都会在游戏中以原生的属性修饰符表示出来，只有少量的特性（例如铜的雷电抗性，金的火焰抗性，钻石的激光抗性）需要你手动查阅，而且它们都很直觉化，几乎不需要解释
