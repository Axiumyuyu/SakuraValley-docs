# 概要

这是一个由多个维度组成的世界，除了我们所熟知的主世界，下界，末地，还存在一些其他维度，这些维度需要在探索的过程中得知其存在，并发现其进入方式。

## 主世界

主世界由两部分组成:

- 在世界中心的数个岛屿，我们一般称为"岛内"或者"中心岛"

- 在这之外的无限世界，我们一般称为"岛外"或者"外岛"

这两部分之间有一片过渡区海洋隔开，海洋内没有任何资源

### 世界范围

主世界高度上限为1024，下限为-256，边界半径为8000

### 资源编排

|      | 岛内         | 岛外            |
|------|------------|---------------|
| 矿石   | ✔          | ✘(仅伴随某些结构生成)  |
| 人造结构 | ✘(极少量)     | ✔             |
| 生成中立/敌对生物   | ✘(无自然生成) | ✔             |
| 生成被动生物   | ✔ | ✔             |
| 能否圈地 | ✔          | ✘(可以但不建议) |

> 需要特别说明的是:
> 
> **岛内默认不允许破坏与放置方块，只有在自己拥有对应权限的领地内才可以放置/破坏方块**
>
> *可以放置流体，但是请勿滥用这个功能，我允许它的本意是防止一些边缘情况导致受困*

简单的理解:岛内是一个用于发展与休憩的地方，岛外是一个用于探索和战斗的地方

### 岛内

中心岛屿地形使用软件生成

在岛内可以使用 `/start` 来在预设的位置中随机传送，如果你正准备寻找一块新的地方建立领地，可以使用这个指令，详细请看[起步](../how-to-start/)

> 请注意，岛内虽然有矿石生成，但这并非你首选的矿物来源，矿物的主要来源仍然是在外岛探索。

### 岛外

岛外的地形由各类数据包生成，含有丰富的人造结构和遗迹，以及和战利品匹配的危险，在这里可以使用`/rtp`来随机传送，传送范围包括岛内和岛外。

### 过渡区

过渡区是将岛内外隔开的平坦海洋，没有任何东西，尽管没有限制移动，但正常情况下你不应该进入这里，如果你想在岛内外之间移动，应该使用领地的传送指令，随机传送或者通过warp设置的传送点。

## 下界

尽管传送指令已经非常便捷，但基岩层上界仍然可以用于搭建设施，并且已经有部分结构会跨越基岩层生成。

### 世界范围

下界的基岩上界高度为 192，高度上限为 320，边界半径 8000。

在下界对应主世界超出边界范围的部分搭建下界传送门返回时，将会被压制在主世界世界边界内部。

## 末地

主岛含有两层的黑曜石柱子，但只有内层会在末影龙复活时生成水晶。

### 世界范围

末地的高度上限为 512，边界半径 8000。
